# Projektregeln

Bei jeder neuen Version muss die Dokumentation zusammen mit dem Code aktualisiert werden.

Pflichtschritte fuer jede Version:

- Version in der Anwendung aktualisieren.
- `CHANGELOG.md` ergaenzen.
- `ROADMAP.md` aktualisieren.
- `MIGRATION.md` aktualisieren.
- `TODO.md` aktualisieren.
- Am Ende eine kurze Zusammenfassung der Aenderungen ausgeben.

Keine Version ohne Dokumentation abschliessen.
